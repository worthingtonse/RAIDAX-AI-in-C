# File System

The root folder is called "Coins." This holds two folders and three files. The Wallet folder holds eleven folders and 
two folders. If these structures are not found, they will be created automatically. 
```C
Coins
  ├── config.toml
  ├── main.log
  ├── simple.log
  ├── Templates
  └── Wallets
    ├── Bank
    ├── config.toml
    ├── Counterfeit
    ├── Export
    ├── Fracked
    ├── Import
    ├── Imported
    ├── Limbo
    ├── Receipts
    ├── Sent
    ├── Suspect
    ├── transactions.csv
    └── Trash
```
